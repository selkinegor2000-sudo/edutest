### 2. 📁 PDF/Word загрузка → AI генерация тестов

**Место:** `/test/generate` → вкладка "Генерировать из файла"  
**Поддерживаемые форматы:** PDF, DOCX, TXT (до 10 МБ)  
**Процесс:**
1. Загрузить документ → парсер извлекает текст
2. Предпросмотр текста + настройка параметров
3. AI создаёт тест на основе содержимого
4. Предпросмотр вопросов + сохранение

**AI генерирует:**
- Правильное распределение типов вопросов (одиночный/множественный/открытый)
- Смешанные уровни сложности (лёгко/средне/сложно)
- Варианты ответов с маркировкой правильного
- Эталонные ответы для открытых вопросов

**Новые зависимости:**
- `pdf-parse` — парсинг PDF
- `pdfjs-dist` — обработка PDF документов
- `mammoth` — парсинг DOCX файлов
- `multer` — обработка загрузок

**Файлы:**
- `server/document-parser.ts` — новый парсер (функции для PDF/Word/TXT)
- `server/routes.ts` — эндпоинты:
  - `POST /api/documents/upload` — загрузка и парсинг файла
  - `POST /api/documents/generate-test` — генерация теста ИИ
- `client/src/pages/test-generate.tsx` — новая страница (3-шаговый wizard)
- `client/src/components/app-sidebar.tsx` — новая вкладка
- `client/src/App.tsx` — маршрут `/test/generate`

### 3. 🤖 Улучшена AI интеграция (альтернативы Groq)

**Проблема:** Groq заблокирован из России  
**Решение:** Гибридный fallback с 3 уровнями

**Уровень 1 — OpenRouter (основной)**
- Использует 4 бесплатные модели с автоматическим fallback:
  - `google/gemma-4-26b-a4b-it:free`
  - `meta-llama/llama-3.3-70b-instruct:free`
  - `qwen/qwen3-next-80b-a3b-instruct:free`
  - `nousresearch/hermes-3-llama-3.1-405b:free`

**Уровень 2 — Yandex GPT (для России)**
- Если OpenRouter не доступен и задан `YANDEX_API_KEY`
- Переменная окружения: `USE_YANDEX=true`

**Улучшения в коде:**
- Более детальное логирование ошибок
- Обработка исключений при недоступности сервиса
- Try-catch блоки для каждого сервиса

**Файлы:**
- `server/routes.ts` — обновлена функция `callAI()` с fallback логикой (70 строк кода)

### 4. 🏆 Достижения и бейджи студентов

**Место:** Страница "Мои результаты" → новая секция достижений  
**Бейджи (автоматические):**
- 🏆 **Отличник** — средний балл ≥ 80%
- ⭐ **Хорошист** — средний балл ≥ 60% но < 80%
- 💯 **Идеально!** — ≥ 1 теста с 100%
- 🎯 **Старательный** — пройдено ≥ 5 тестов
- 🔥 **Мастер** — пройдено ≥ 10 тестов
- 🌟 **Без ошибок** — все тесты сданы на ≥60%

## 🚀 Использование

### Для преподавателя:

**1. Просмотр аналитики:**
```
Сайдбар → Аналитика → графики и статистика студентов
```

**2. Создание теста из документа:**
```
Сайдбар → Генерировать из файла
1. Загрузить PDF/DOCX/TXT
2. Указать параметры (название, предмет, уровень сложности)
3. ИИ создаёт тест
4. Сохранить в систему
```

### Для студента:

**1. Просмотр достижений:**
```
Сайдбар → Мои результаты → Секция "Ваши достижения"
(бейджи появляются автоматически при выполнении условий)
```

**2. Фильтрация истории:**
```
Мои результаты → Нажать на кнопку предмета для фильтрации
```
